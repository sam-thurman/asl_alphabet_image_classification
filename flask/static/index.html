ASL Image Classifier

This project serves as proxy to a Convolutional Neural Network that can detect and interpret ASL from images and live video. The model used in this project was built from scratch using the Keras Functional API (running Tensorflow 2.1.0 back-end). CNN was trained on 87000 jpg images of hands, 3000 images per letter of the American Sign Language alphabet, as well as 3000 images for 'delete', 'space', and 3000 images of blank rooms categorized as 'nothing'. Image data can be found and downloaded [here](https://www.kaggle.com/grassknoted/asl-alphabet). The goal of this project was to gain a deeper understanding of Convolutional Neural Network architecture, as well as to explore real time classification using computer vision. Due to the initial time constraints on deployment, currently this page serves purely as a demonstration of the model's performance on the provided validation data. You can explore how the model performs classifying a live video feed by cloning down the repository for this project, and following the instructions in the Readme.md filename
